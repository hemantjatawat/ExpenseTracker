# Expense Tracker Project

## Project Description
This is a Java-based Expense Tracker application that helps users record and manage their daily expenses. 
The project allows users to add expenses, view all expenses, and store data securely using a MySQL database.

## Features
- Add daily expenses
- View expense history
- Categorize expenses
- Console-based user interface
- MySQL database integration

## Technologies Used
- Java
- JDBC
- MySQL
- Eclipse IDE

## Project Structure
- src/
  - db/ (Database connection)
  - model/ (Expense model)
  - dao/ (Database operations)
  - main/ (Main application)
- database/
  - expense_tracker.sql

## How to Run the Project
1. Clone or download the repository.
2. Import the project into Eclipse.
3. Create the database using `expense_tracker.sql`.
4. Update database credentials in `DBConnection.java`.
5. Run `MainApp.java`.
